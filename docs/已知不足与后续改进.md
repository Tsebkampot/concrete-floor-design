# 已知不足与后续改进

## 一、已知不足

1. 最不利内力包络采用课程设计简化控制点方法，不替代教材系数表或精确结构分析。
2. 主梁按集中力系数计算，未自动建立完整多跨连续梁刚度矩阵。
3. 斜截面箍筋仅估算 `Av/s`，未自动检查所有规范构造条文。
4. 抵抗弯矩图为示意图，未自动确定钢筋截断、弯起和锚固长度。
5. 程序不直接导出 PDF，当前支持 Word、Excel、Markdown 和 PNG。
6. 中文图形字体依赖本机环境，部分电脑可能出现字体替换。

## 二、人工复核要求

- 程序结果必须与课程设计手算过程对比。
- 弯矩系数、剪力系数、活载布置应按教材或教师要求调整。
- 配筋方案应结合最小配筋率、钢筋净距、保护层、锚固和构造要求。
- 计算书最终提交前应检查单位、有效高度、支承条件和构件编号。

## 三、后续改进方向

- 增加连续梁精确内力求解和影响线分析。
- 增加规范最小配筋率、裂缝、挠度和耐久性校核。
- 增加构件编号管理和完整计算书模板。
- 增加 PDF 一键导出。
- 增加更多教材算例并扩大自动化测试覆盖。
