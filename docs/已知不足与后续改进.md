# 已知不足与后续改进

## 一、当前分析边界

1. 板按 1 m 宽单向板带、次梁和主梁按 Euler-Bernoulli 连续梁分析，未建立楼盖二维板壳或空间框架模型。
2. EI 默认采用毛截面 I=bh³/12 并乘用户输入的刚度折减系数；未自动迭代开裂刚度和长期效应。
3. 活载对最多 8 跨枚举全部逐跨组合，未进行移动荷载连续影响线分析。
4. 常规楼盖按代表性内支座反力完成板→次梁→主梁传递；不规则楼盖应逐根建立实际拓扑。
5. 最小纵筋率暂按课程设计默认值 0.2% 筛选，最大配筋、裂缝、挠度、耐久性和抗震构造尚未完整自动化。
6. 箍筋仍采用课程设计简化抗剪估算；锚固、截断、加密区和节点构造必须人工复核。

## 二、人工复核要求

- 检查跨数、逐跨跨度、支承条件、支座宽度、E 和刚度折减系数。
- 检查代表性反力是否适用于实际板区及构件编号。
- 按课程指定教材或规范复核最小配筋、箍筋、净距、保护层、锚固和裂缝挠度。
- 将程序内力与至少一个手算或权威结构分析算例对照。

## 三、后续方向

- 增加逐根板带/梁拓扑映射和不规则布置。
- 增加开裂刚度迭代、裂缝、挠度、耐久性和完整规范条文校核。
- 增加纵筋截断、锚固、配筋立面及截面详图自动绘制。
- 增加影响线和移动荷载分析。
