# 公式与适用范围

## 一、荷载计算公式

### 1. 板恒载

```text
g1 = h / 1000 × γc
g2 = t / 1000 × γm
gk = g1 + 水磨石面层荷载 + g2
```

其中 `h`、`t` 输入单位为 mm，计算时换算为 m；`γc`、`γm` 单位为 kN/m3；结果单位为 kN/m2。

### 2. 荷载设计值

```text
gd = γ0 × γG × gk
qd = γ0 × γQ × qk
```

`γ0` 为结构重要性系数，默认 1.0。

### 3. 梁自重和粉刷

```text
次梁自重 = γc × b × (h - hf)
次梁粉刷 = γm × t × (h - hf) × 2
主梁自重线荷载 = γc × b × (h - hf)
主梁粉刷线荷载 = γm × t × (h - hf) × 2
```

梁自重只计算板下梁肋高度 `h - hf`，避免重复计入板自重。

## 二、荷载传递公式

```text
板矩阵分析支座反力（1 m 板带）kN = 次梁线荷载 kN/m
次梁矩阵分析支座反力 kN = 主梁交点集中力 kN
```

正式传力不再选取或复制“代表性最大反力”。对每个全局工况，程序先求板带各支座反力，并除以 `strip_width_m` 换算为板支承线单位线反力（kN/m）；随后逐支承线建立次梁计算，再把对应次梁支座反力按主梁交点位置施加为节点集中力（kN）。G、Q、G+Q 和逐跨活载工况名称在三级结果表中保持一致。原“分担宽度”和“线荷载×跨度”只保留在“手算对比附录”，明确标注为非正式结果。

当前逐线映射适用于规则等跨楼盖，并要求板跨总长等于单个主梁跨度、次梁跨长总和等于 L2、主梁跨长总和等于 L1。不规则布置会直接报错并提示人工复核。

## 三、矩阵刚度法

板按 1 m 宽板带、次梁和主梁均使用两节点 Euler-Bernoulli 梁单元。节点自由度为竖向位移 v 和转角 θ，单元自由度为 [vi, θi, vj, θj]。

```text
K d = F
fe = ke de - feq
```

单元刚度矩阵采用 EI/L³ 乘以题目规定的标准 4×4 矩阵。均布荷载采用一致节点荷载；主梁集中力位置拆分为节点。求解节点位移后恢复杆端力，并按单元平衡关系重构任意位置 M(x)、V(x)。

统一使用 kN-m 单位制：竖向位移向上为正，转角逆时针为正，输入竖向荷载向下为正；截面正弯矩表示跨中板底/梁底受拉。

旧的 M=αqL²、V=βqL、M=αPL 只保留在手算对比代码中，不参与正式多跨设计结果。

## 控制截面和配筋判定

控制截面同时包括边跨/中跨跨中、内支座中心、内支座左右真实边缘、剪力控制截面和跨内 V=0 弯矩极值点。支座边缘采用输入的真实支座宽度，不再截断为 0.2L；净跨小于等于 0 或支座边缘重叠时直接报错。结果表分别保留支座中心弯矩与支座边缘弯矩：中心截面用于检查最大负弯矩，边缘截面用于面内弯矩和剪力检查。

梁跨中正弯矩先按 T 形截面判断第一类/第二类，支座负弯矩按矩形截面。`bf'` 可输入；输入 0 时暂按 `b+12hf` 的课程近似值估算。程序同时检查 `x≤ξb h0`、纵筋面积和基本可布置性。最小/最大配筋率、保护层、钢筋净距、箍筋直径与最大间距集中在 `calculations/specification_parameters.py`。这些取值及抗剪公式均标注“按课程近似公式估算，需人工复核”，不能冒充完整规范设计。

## 四、配筋计算公式

程序采用单筋矩形截面简化关系：

```text
M = fc × b × x × (h0 - x/2)
As = fc × b × x / fy
```

弯矩单位换算：

```text
1 kN·m = 1,000,000 N·mm
```

## 五、配筋推荐和超配率

### 1. 板筋

```text
As_provided = 单根钢筋面积 × 1000 / 间距
```

### 2. 梁纵筋

```text
As_provided = 根数 × 单根钢筋面积
```

### 3. 超配率

```text
超配率 = (实配面积 - 计算所需面积) / 计算所需面积 × 100%
```

超过 30% 时，程序提示“可能偏保守，建议复核”。

## 六、活载组合与包络

程序提供以下活载布置：

- 全跨活载
- 奇数跨活载
- 偶数跨活载
- 单跨活载
- 相邻跨活载

程序对最多 8 跨枚举 G 与各非空活载跨子集，每个组合均重新调用矩阵求解器。控制截面分别记录最大/最小弯矩、剪力及对应工况；连续图线来自单元内力函数，不使用三点二次插值。

## 七、抵抗弯矩图简化假定

程序根据推荐纵筋估算抗弯承载力：

```text
x = fy × As / (fc × b)
Mu = fy × As × (h0 - x/2)
```

抵抗弯矩图只显示支座负筋范围和跨中正筋范围，纵筋截断、弯起、锚固长度需人工确定。

## 八、适用范围

- 适用于《水工钢筋混凝土》课程设计中的整体式单向板肋形楼盖辅助计算。
- 适用于常规板、次梁、主梁的荷载传递、内力和配筋初步计算。
- 适用于答辩演示、计算书整理和手算对比。

## 九、需要人工复核的内容

- 梁模型、支承条件、EI 和刚度折减系数是否与课程假定一致。
- 活载最不利布置和包络值是否与手算一致。
- 箍筋最小构造、最大间距和加密区。
- 纵筋锚固、截断、弯起位置。
- 裂缝、挠度、耐久性和最小配筋率。
- 构件编号、计算跨度、支承条件和构造详图。
