# 程序说明书

## 一、程序名称

`concrete_floor_design`：整体式单向板肋形楼盖设计辅助计算平台。

## 二、开发工具和运行环境

- 编程语言：Python 3
- 网页界面：Streamlit
- 表格处理：Pandas
- 图形绘制：标准库生成 SVG/PNG
- Word 导出：python-docx
- Excel 导出：openpyxl
- 测试工具：pytest

## 三、程序功能简介

程序由单一 Streamlit 入口 `app.py` 组织成多页面应用，包含首页、基本参数输入、截面初估、板计算、次梁计算、主梁计算、荷载传递、内力包络、配筋推荐、抵抗弯矩图、智能校核、结果导出和程序说明。

所有基础参数保存在 `st.session_state["params"]`，板、次梁、主梁、导出和校核模块共用同一套参数。

## 五、计算流程

1. 输入统一基本参数。
2. 程序计算板自重、面层、抹面，汇总板面恒载标准值。
3. 将 1 m 板带建立为多跨连续梁，逐工况求板支座反力。
4. 板的代表性矩阵反力作为次梁线荷载，叠加次梁自重和粉刷。
5. 次梁矩阵反力按次梁间距施加在主梁实际交点，集中力位置自动拆分节点。
6. 恒载作用于全部跨，活载按逐跨组合枚举并形成正负内力包络。
7. 自动识别支座边缘、跨中、真实弯矩极值点和集中力两侧控制截面。
8. 每个控制截面分别计算正、负弯矩配筋和剪力箍筋。
9. 程序生成控制截面示意图、真实分段弯矩图和剪力包络图。
10. 智能校核模块检查单位、参数范围、截面初估、结果完整性和人工复核提示。
11. 导出 Excel、Word、PDF、Markdown 和 PNG。

## 六、主要公式和计算依据

### 1. 板荷载

```text
板自重 g1 = h / 1000 × γc
抹面荷载 g2 = t / 1000 × γm
恒载标准值 gk = g1 + 水磨石面层 + g2
恒载设计值 gd = γ0 × γG × gk
活载设计值 qd = γ0 × γQ × qk
板带线荷载 q = (gd + qd) × 板带宽度
```

### 2. 矩阵刚度法

```text
单元自由度：[vi, θi, vj, θj]
总体方程：K d = F
单元内力：fe = ke de - feq
```

单元刚度矩阵为 `EI/L³` 乘以：

```text
[ 12,  6L, -12,  6L]
[ 6L, 4L², -6L, 2L²]
[-12, -6L,  12, -6L]
[ 6L, 2L², -6L, 4L²]
```

程序使用 kN-m 单位制。位移向上为正、转角逆时针为正；截面正弯矩表示板底或梁底受拉。经验内力系数只用于手算参考，不参与正式结果。

正式分析按同一全局工况依次执行“板求解 → 板支承线反力除以板带宽度 → 次梁求解 → 次梁交点反力传主梁 → 主梁求解”。程序显示整体竖向力平衡、整体力矩平衡、自由节点竖向力和弯矩残差。当前逐线映射只支持规则等跨楼盖，不支持时会明确报错。

控制表同时保存支座中心和真实支座边缘内力；支座中心用于负弯矩检查，边缘用于面内弯矩和剪力检查。网页、PNG、Word、PDF、Excel 同时输出板、次梁 30m 方向和主梁 18m 方向的控制截面示意图，图中编号与结果表共用同一组实际提取位置。

梁跨中正弯矩按 T 形截面判断，支座负弯矩按矩形截面。规范参数和抗剪公式尚未完成特定规范版本的逐条复核，输出统一标注“按课程近似公式估算，需人工复核”。

### 3. 正截面配筋

```text
1 kN·m = 1,000,000 N·mm
M = fc × b × x × (h0 - x/2)
As = fc × b × x / fy
```

### 4. 斜截面箍筋简化估算

```text
Vc = 0.7 × sqrt(fc) × b × h0
Av/s = max(V - Vc, 0) / (fyv × h0)
```

这是课程设计辅助估算，箍筋最小构造、最大间距、加密区要求应按教材或规范人工复核。

## 七、输出结果说明

- 荷载表：列出标准值、设计值、公式和单位。
- 内力表：列出每个控制截面的正负弯矩、正负剪力、反力及控制工况。
- 配筋表：每个控制截面独立列出所需面积、最小面积、推荐方案、利用率和箍筋。
- 图形：控制截面示意图、矩阵弯矩包络图和剪力包络图，均支持 PNG 导出。
- 校核表：按错误、警告、提示三级显示。
- 计算书：支持 Excel、Word 和 Markdown 导出。

## 八、测试算例

程序内置测试采用课程设计示例数据：

## 九、程序结果与手算结果对比

程序采用完整小数计算，手算和课程设计示例中常在中间步骤四舍五入，因此结果会出现小范围差异。只要差异来自小数保留位数，且公式和单位一致，可在误差说明中注明。

## 十、已知不足和后续改进方向

- 采用线弹性 Euler-Bernoulli 梁模型，未考虑开裂后的非线性刚度重分布。
- 活载通过有限跨组合枚举，不是移动荷载连续影响线分析。
- 抵抗弯矩图仅做示意，未自动确定纵筋截断、弯起和锚固长度。
- 箍筋计算为简化估算，最小配箍率、加密区和构造要求需人工复核。
- 最小纵筋率默认取 0.2%，应按课程采用的具体规范复核。
