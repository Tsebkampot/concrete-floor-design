# 测试算例与手算对比

## 一、测试目的

验证程序在课程设计示例参数下，能正确完成板、次梁、主梁的荷载计算、荷载传递和基本设计值计算，并说明程序结果与手算结果之间的误差来源。

## 二、板计算对比

### 1. 示例参数

| 参数 | 数值 | 单位 |
|---|---:|---|
| 水磨石面层 | 0.65 | kN/m2 |
| 板厚 | 80 | mm |
| 钢筋混凝土重度 | 25 | kN/m3 |
| 石灰砂浆抹面厚度 | 15 | mm |
| 石灰砂浆重度 | 17 | kN/m3 |
| 活荷载标准值 | 4 | kN/m2 |
| 恒载分项系数 | 1.05 | - |
| 活载分项系数 | 1.2 | - |
| 板带宽度 | 1 | m |

### 2. 手算过程

```text
80 mm 钢筋混凝土板自重 = 0.080 × 25 = 2.000 kN/m2
15 mm 石灰砂浆抹面 = 0.015 × 17 = 0.255 kN/m2
恒载标准值 gk = 2.000 + 0.650 + 0.255 = 2.905 kN/m2 ≈ 2.91 kN/m2
恒载设计值 gd = 1.05 × 2.905 = 3.05025 kN/m2
活载设计值 qd = 1.2 × 4 = 4.800 kN/m2
板带线荷载设计值 q = (3.05025 + 4.800) × 1 = 7.85025 kN/m
```

课程设计中若中间值取 `2.91 kN/m2`，则：

```text
q = 1.05 × 2.91 + 1.2 × 4 = 7.8555 kN/m ≈ 7.856 kN/m
```

### 3. 程序结果

| 项目 | 程序结果 | 手算近似值 | 说明 |
|---|---:|---:|---|
| 板自重标准值 | 2.000 | 2.000 | 一致 |
| 抹面荷载标准值 | 0.255 | 0.255 | 一致 |
| 恒载标准值 | 2.905 | 2.91 | 四舍五入差异 |
| 恒载设计值 | 3.05025 | 3.056 | 取值精度差异 |
| 活载设计值 | 4.800 | 4.800 | 一致 |
| 板带线荷载设计值 | 7.85025 | 7.856 | 取值精度差异 |

## 三、次梁计算对比

### 1. 手算过程

```text
板传来恒载 = 2.91 × 2 = 5.82 kN/m
次梁自重 = 25 × 0.2 × (0.4 - 0.08) = 1.6 kN/m
次梁粉刷 = 17 × 0.015 × (0.4 - 0.08) × 2 = 0.1632 kN/m
恒载标准值 gk = 5.82 + 1.6 + 0.1632 = 7.5832 kN/m ≈ 7.583 kN/m
恒载设计值 gd = 1.05 × 7.5832 = 7.96236 kN/m ≈ 7.962 kN/m
活载标准值 qk = 4 × 2 = 8 kN/m
活载设计值 qd = 1.2 × 8 = 9.6 kN/m
```

### 2. 程序结果

| 项目 | 程序结果 | 手算结果 | 说明 |
|---|---:|---:|---|
| 板传来恒载 | 5.82 | 5.82 | 一致 |
| 次梁自重 | 1.60 | 1.60 | 一致 |
| 次梁粉刷 | 0.1632 | 0.1632 | 一致 |
| 恒载标准值 | 7.5832 | 7.583 | 四舍五入差异 |
| 恒载设计值 | 7.96236 | 7.962 | 四舍五入差异 |
| 活载标准值 | 8.00 | 8.00 | 一致 |
| 活载设计值 | 9.60 | 9.60 | 一致 |

## 四、主梁计算对比

### 1. 手算过程

```text
次梁传来恒载集中力 = 7.583 × 6 = 45.498 kN
主梁自重 = 25 × 0.3 × (0.65 - 0.08) × 2 = 8.55 kN
主梁粉刷 = 17 × 0.015 × (0.65 - 0.08) × 2 × 2 = 0.5814 kN
恒载标准值 Gk = 45.498 + 8.55 + 0.5814 = 54.6294 kN ≈ 54.629 kN
恒载设计值 Gd = 1.05 × 54.6294 = 57.36087 kN ≈ 57.36 kN
活载标准值 Qk = 4 × 6.0 × 2.0 = 48 kN
活载设计值 Qd = 1.2 × 48 = 57.6 kN
```

### 2. 程序结果

| 项目 | 程序结果 | 手算结果 | 说明 |
|---|---:|---:|---|
| 次梁传来恒载集中力 | 45.498 | 45.498 | 一致 |
| 主梁自重 | 8.55 | 8.55 | 一致 |
| 主梁粉刷 | 0.5814 | 0.5814 | 一致 |
| 恒载标准值 | 54.6294 | 54.629 | 四舍五入差异 |

## 五、荷载传递对比

| 传递步骤 | 手算公式 | 程序公式 | 结果 |
|---|---|---|---:|
| 板恒载到次梁 | 2.91 × 2 | 板面恒载标准值 × 次梁间距 | 5.82 kN/m |
| 板活载到次梁 | 4 × 2 | 板面活载标准值 × 次梁间距 | 8.00 kN/m |
| 次梁恒载到主梁 | 7.583 × 6 | 次梁恒载线荷载 × 次梁跨度 | 45.498 kN |
| 次梁活载到主梁 | 8 × 6 | 次梁活载线荷载 × 次梁跨度 | 48.00 kN |

## 六、误差来源说明

1. 板恒载标准值可按 `2.905 kN/m2` 继续计算，也可按课程设计中间步骤取 `2.91 kN/m2`，两者会造成约 `0.006 kN/m` 的线荷载差异。
2. 程序默认保留完整小数，显示表格再统一四舍五入。
3. 旧手算内力系数仅用于教学对照，正式程序弯矩和剪力来自矩阵刚度法。
4. 箍筋估算、抵抗弯矩和构造设计仍不能替代规范复核。

## 八、矩阵刚度法基准算例

新增测试使用解析解核对矩阵求解器：

| 算例 | 核对值 |
|---|---|
| 简支梁均布荷载 | 两端反力 qL/2，跨中弯矩 qL²/8 |
| 简支梁跨中集中力 | 两端反力 P/2，跨中弯矩 PL/4，集中力两侧剪力发生 P 的跃变 |
| 固端梁均布荷载 | 两端负弯矩绝对值 qL²/12 |
| 对称两跨连续梁 | 两端反力及镜像位置弯矩满足对称性 |
| 总体方程 | 总刚度矩阵对称，支座反力与外荷载满足竖向平衡 |

前文的“面荷载×分担宽度”和“线荷载×跨度”表格仅作为旧课程手算荷载对比。正式多跨程序使用板和次梁的矩阵支座反力逐级传递。

当前自动测试还覆盖：

- 支座边缘、跨中、实际弯矩极值点和集中力两侧编号。
- 每个控制截面独立配筋且实配面积不小于设计面积。
- 控制截面示意图编号与结果表一致。
