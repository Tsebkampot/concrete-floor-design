# 测试算例与手算对比

本文件记录程序内置测试与课程设计手算对照口径，便于提交前复核。

## 自动化测试

项目测试位于 `tests/` 目录，建议提交前运行：

```bash
python3 -m pytest -q
```

当前测试覆盖：

- 板、次梁、主梁基础荷载计算与课程示例值对比。
- 简支梁、固端梁、两跨连续梁等矩阵刚度法基准算例。
- 板支承线反力传递、次梁传主梁集中力、全局工况连续性。
- 控制截面编号、支座中心与真实支座边缘、逐截面配筋。
- Excel、Word、Markdown、PDF 导出结构回归。
- 自动计算书只输出汇总表，不倾倒全部矩阵明细。

## 手算对照口径

程序保留“系数法对照”模块，用于复现小组表中板、次梁、主梁的课程设计口径。该模块独立于正式矩阵刚度法链路，主要用于答辩和计算书对账。

主梁部分同时保留两个恒载口径：

- `300×600修正口径`：按当前主梁截面尺寸重算恒载。
- `截图原始口径（复现补充页）`：用于复现补充截图中的主梁弯矩、剪力和 As。

最终计算书提交前应统一主梁恒载、内力和配筋口径，并人工复核截图原始数据与 300×600 修正数据的差异。

## 人工复核边界

自动结果不能替代课程设计手算、教材系数表和教师要求。以下内容需人工复核：

- 主梁 300×600 截面对应的恒载、支座负弯矩、剪力包络和配筋面积。
- T 形截面翼缘有效宽度、受压区高度限值和截面尺寸条件。
- 斜截面抗剪、最小配筋率、箍筋间距、锚固长度、钢筋截断和构造钢筋。
- 不规则楼盖、非等跨布置或教师指定特殊工况。
